# Mini projekt — Raisa Rachynska

## Dataset

https://www.kaggle.com/datasets/desolution01/messy-employee-dataset

## Opis

Wybrałam dataset „Messy Employee Dataset”. Jest to syntetyczny zbiór danych pracowników zawierający brakujące wartości, duplikaty oraz niespójne dane.

## Cel projektu

Celem projektu jest oczyszczenie danych, wykonanie analizy eksploracyjnej (EDA) oraz budowa modelu predykcyjnego.

## Target

Jako zmienną docelową planuję wykorzystać kolumnę Performance_Score.
Chcę sprawdzić, czy na podstawie cech pracownika można przewidzieć jego ocenę.
